Instalacin de Paquetes

Hay varias formas de instalar paquetes. Hoy te cuento una con la que podes instalar y llamar varios paquetes juntos!

Si te resulta ms fcil, podes copiar y pegar los pasos en todos tus trabajos.

Te muestro el proceso de 3 pasos

1 Paso

Para realizar la instalacin y el llamado de los paquetes debemos correr (Run) la funcin ipak()1

ipak <- function(pkg){
  new.pkg <- pkg[!(pkg %in% installed.packages()[, "Package"])]
  if (length(new.pkg)) 
    install.packages(new.pkg, dependencies = TRUE)
  sapply(pkg, require, character.only = TRUE)
}

2 Paso

Creamos un objeto llamado packages(), el cual contiene los diversos paquetes a utilizar.

Los paquetes los escribimos entre comillas y separados por comas:

packages <- c("tidyverse","cluster","tidyr")

3 Paso

Por ltimo debemos correr (Run) la linea ipak(), con esto ya se realizar el proceso de instalacin y de llamado de los paquetes detallados en el paso 2:

ipak(packages)
## tidyverse   cluster     tidyr 
##      TRUE      TRUE      TRUE
